Map Boundary Builder

A local CLI and browser workspace for extracting georeferenced service-area polygons from map screenshots.

View Source Quick Start

Local Pipeline

Built for repeatable boundary extraction.

  1. 01 Detect the fill

    Segments the service area across light, green, bright-blue, and dark map styles.

  2. 02 Fit the map

    Uses OCR labels, geocoding, and OpenStreetMap context to infer a transform.

  3. 03 Export GeoJSON

    Writes the polygon, confidence metadata, mask overlay, and copyable output.

Run It

Bring your own map crop.

python3 -m venv .venv
.venv/bin/python -m pip install -e .
.venv/bin/map-boundary --image /path/to/map.png --city "Las Vegas" --output out/boundary.geojson
